# EntrenamientoYOLO_P-C
Este directorio esta diseñado para la orientación o ayuda para el entrenamiento de modelos de detección YOLO, partiendo desde el YOLOV3 hasta el YOLOV8

#Script_preparacion
  * En esta carpeta se encuentra unicamente el archivo para la preparación de las imagenes, este archivo divide en un dataset de imágenes (y sus etiquetas correspondientes en formato .txt) en carpetas separadas para entrenamiento y validación, además de generar dos archivos de texto (train.txt y val.txt) con las rutas de las imágenes para cada subconjunto.
    
#Script_entrenamiento
  * En esta carpeta se encuentran todas las subcarpetas de entrenamiento del YOLOV3 hasta el YOLOV8, cada uno cuenta con el script de entrenamiento
